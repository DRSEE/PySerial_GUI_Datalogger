Ce code permet de lire les données d'un port série en temps réel ainsi que d'exporter les données visualisé sous format .csv ou .txt,
Codé en Python l'interface graphique est crée via Tktinter, et la lecture de port série et géré par PySerial

Le code tourne avec la version Python 3.7.1 

Fonctionne sous W10 autre OS pas testé
